#GitHub+hexo 个人博客搭建教程 （Ubuntu）

##1.安装nodejs和npm
node.js的官网链接下载tar.xz的包，解压并安装

tar -xJf node-v12.3.1-linux-x64.tar.xz

设置软连接

ln -s /opt/node-v12.3.1-linux-x64/bin/node /usr/local/bin/node 
ln -s /opt/node-v12.3.1-linux-x64/bin/npm /usr/local/bin/npm

查看相应node和npm的版本，如果可以查看版本，说明安装成功

node -v  &&  npm -v

##2.安装hexo和初始化配置
创建目录及安装hexo

mkdir hexo 目录名称
cd hexo 目录名称
npm install -g hexo-cli

设置软连接

ln -s /opt/node-v12.3.1-linux-x64/bin/hexo /usr/local/hexo

查看版本及启动hexo

hexo -v
hexo server

如显示hexo is running at http://localhost:4000.说明本地博客搭建成功

##3.部署到GitHub
创建项目仓库
登录Github官网，点击右上角的+，选择New repository。
在页面里输入github账户名.github.io只能这么填，不能改，例如我的是xxx.github.io填完后点击Create repository即可。

配置本机全局git环境

git config --global user.email"xxx.@.xxxx.com"
git config --global user.name"your name"

生成ssh密钥和查看密钥，将密钥导入到GitHub

ssh-keygen
gedit ~/.ssh/id.pub

github首页右上角个人头像点击，选择settings，左列表选择SSH and GPG keys，点击New SSH key.

测试ssh

ssh -T git@github.com

##4.配置hexo，编译和部署
博客所在目录，找到_config.yml,配置_config.yml

deploy:
  type: git
  repo: git@github.com:xxxx/xxxx/gitgub.io.git
  branch: master

编译，上传代码

hexo generate
hexo deploy

如果未出现任何错误，访问xxx@github.io,说明博客已搭建完成。
若出现 error deployer not found: git 解决方法：

npm install --save hexo-deployer-git

重复上面步骤即可。

以上就是搭建博客的步骤
